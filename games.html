The Crew 2 é um jogo eletrônico de corrida, desenvolvido pela Ivory Tower e publicado pela Ubisoft para Microsoft Windows, PlayStation 4 e Xbox One. É a sequencia de seu título antecessor, The Crew de 2014.

Possui um ambiente de mundo aberto persistente[nota 2] para vagar livremente em uma escala reduzida do mapa dos Estados Unidos.

O jogo permite que os jogadores controlem uma variedade de veículos, incluindo carros, motocicletas, barcos e aviões. O jogo foi lançado em 29 de junho de 2018.






MINIMUM CONFIGURATION - 30 FPS:

  • OS: Windows 7 SP1, Windows 8.1, Windows 10 (64-bit versions only)
  • PROCESSOR: Intel Core i5-2400s @ 2.5GHz or AMD FX-6100 @ 3.3GHz or equivalent*
  • VIDEO CARD: NVIDIA GeForce GTX 660 or AMD HD 7870 (2GB VRAM with Shader Model 5.0 or better)
  • SYSTEM RAM: 8GB

RECOMENDED CONFIGURATION - 60 FPS:

  • OS: Windows 7 SP1, Windows 8.1, Windows 10 (64-bit versions only)
  • PROCESSOR: Intel Core i5-3470 @ 3.2 GHz or AMD FX 6350 @ 3.9 GHz
  • VIDEO CARD: Nvidia GeForce GTX 760 or AMD R9 270X (4GB / 2GB V
  • RAM: 8GB
